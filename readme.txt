This Wordpress plugin will block IE6 users with a choice to upgrade.

== Description ==

This plugin keeps people from viewing your site if they are using Internet Explorer 6. Gives options to download a variety of newer browsers.

For more information on the plugin visit: http://vasthtml.com/js/wordpress-block-ie6-plugin/

== Installation ==

This section describes how to install the plugin and get it working.
 
1. Upload `block-ie6` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Please visit http://vasthtml.com/js/wordpress-block-ie6-plugin/ for usage instructions.
